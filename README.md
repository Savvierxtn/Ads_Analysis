# Marketing Analytics Project

## Overview

This project integrates Google, Facebook, and TikTok ads data into a
unified model.

## Objectives

-   Combine multi-platform data
-   Standardize metrics
-   Generate insights

## Data Sources

-   Google Ads
-   Facebook Ads
-   TikTok Ads

## Metrics

CTR, CPC, CPA, CVR, CPM, Funnel Efficiency

## Insights

-   Platform performance comparison
-   Budget optimization opportunities

## Tools

-   PostgreSQL / Supabase
-   SQL
-   Looker Studio

## Conclusion

Unified analytics enables better marketing decisions.
